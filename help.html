Steps to Export Test results to a TVTS import JSON file

  1. Execute the tests and fill in the results in column G
  2. From the TVTS menu, run the Export command
  3. Place the downloaded JSON file in the TVTS root folder
  4. The JSON file is used as input to the TVTS test "TvtsSmokeTestCases".
  5. Run TvtsSmokeTestCases from TVTS by one of the following ways:
    • The TvtsSmokeTestCases test will run as part of the TVTS certification tests, or
    • Run TvtsSmokeTestCases standalone by executing the following command:
      • tvts-tradefed run commandAndExit tvts-cert -m TvtsSmokeTestCases
    • Run TvtsSmokeTestCases standalone and override JSON folder location:
      • add option: --module-arg TvtsSmokeTestCases:set-option:input-folder:"/home/tester/Downloads"
      • Where "/home/tester/Downloads" corresponds to the folder containing your JSON files
